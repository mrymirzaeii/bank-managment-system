Bank Management System

Advanced Programming Final Project

Project Description

This project is a simple banking management system developed in Python for the Advanced Programming course.
The system supports two types of users: Customers and Bankers.
Data is stored using CSV files and the project is implemented using Object-Oriented Programming (OOP).

User Roles

Customer

Customers can:

- Login to the system
- View their accounts
- Deposit money
- Withdraw money
- Transfer money to another account

Banker

Bankers can:

- Create new accounts for existing customers
- Add new customers
- View all accounts
- Block or unblock accounts
- View all users
- View transaction reports

Features

- User Authentication
- Account Management
- Deposit and Withdrawal Operations
- Money Transfer Between Accounts
- Transaction History Recording
- Transaction Reporting
- Account Blocking and Unblocking
- Savings Account Withdrawal Restrictions
- CSV File Storage
- Object-Oriented Programming (OOP)
